## 🎯 Introduction

**FocusDuo** is a native macOS productivity app designed to help you manage complex mental priorities by keeping your **top 2 tasks** always visible and front and center. Stop getting lost in multitasking chaos!

This is NOT just another TODO app—it's a brain-sync tool that visualizes your priorities and keeps you focused on what truly matters.

---

## ✨ Core Features

### 🎯 Visual Priority Management
- **Focus on Top 2**: Pin your most important tasks at the top
- **Drag & drop** reordering for instant priority changes
- **Always on top** window mode for persistent visibility
- **Minimal mode** for distraction-free focus (Premium)

### ⏱️ Pomodoro Timer
- Built-in focus timer with customizable durations (5m, 25m, 45m, 60m)
- **7 sound themes**: Chime, Bell, Success, Gentle, Alert, Zen, Classic
- Visual progress indicator
- Desktop notifications when timer completes

### 💎 Premium Features ($4.99)
- **Edit tasks** anytime with full text editing
- **Import/Export** data in JSON format
- **Up to 50 tasks** (vs 3 in free version)
- **Minimal mode** UI for maximum focus
- **Advanced settings** and customization
- **Time tracking** display
- All future updates included

### 🆓 Free Version
- Up to **3 tasks**
- Focus on top 2 tasks
- Drag & drop ordering
- Always on top mode
- Full Pomodoro timer
- All basic features

## 📥 Installation

### System Requirements
- **macOS 10.15** (Catalina) or later
- **100 MB** disk space
- Internet connection for license activation (one-time)

### Installation Steps
1. Download **FocusDuo-1.0.0.dmg** from Assets below
2. Open the DMG file
3. Drag **FocusDuo** to your Applications folder
4. Launch from Applications (⌘+Space → "FocusDuo")
5. Grant necessary permissions if prompted

### First Launch
- The app will appear in the **top-right corner** of your screen
- Try adding your first task!
- Access settings via the ⚙️ button

## 🔄 Auto-Update

FocusDuo includes an automatic update checker:
- Checks for updates **every time** you launch the app
- Downloads updates in the background (with your permission)
- Notifies you when updates are ready
- One-click install and restart
